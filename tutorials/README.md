# Six-Week R Data Analysis Tutorials

Use these files in RStudio as hands-on practice.

## How to Use

1. Open RStudio.
2. Open one tutorial script from this folder.
3. Run code line by line with `Ctrl+Enter` on Windows or `Cmd+Enter` on Mac.
4. Read the comments, then complete each `TRY IT` section.

Install the main packages once:

```r
install.packages("tidyverse")
install.packages("nycflights13")
```

If you only have time for one thing each week, finish the `CHECKPOINT` section in that week's script.

## Tutorial Files

1. `week-01-setup-orientation.R` - RStudio, scripts, CSV loading, basic inspection
2. `week-02-dplyr-data-manipulation.R` - pipe, filter, select, arrange, mutate, summarise
3. `week-03-ggplot2-visualization.R` - common chart types and faceting
4. `week-04-reshape-join-import.R` - pivoting, joins, dates, CSV import
5. `week-05-project-starter.R` - project structure and guided exploratory analysis
6. `week-06-quarto-communication.R` - analysis narrative and report preparation

## Suggested Workflow

Create a small notebook or document where you answer the checkpoint questions each week. By Week 6, those answers become the first draft of your project report.
