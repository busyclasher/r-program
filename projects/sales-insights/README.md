# Sales Insights Project

This project analyzes the sample sales data in `../../data/sales.csv`.

It creates:

- Summary tables for revenue by region, category, and month
- Three chart images
- A short text report with key metrics

## Run It

From the repository root:

```powershell
Rscript projects/sales-insights/sales-insights.R
```

## Output

After running the script, check:

```text
projects/sales-insights/output/
```

The folder will contain CSV files, PNG charts, and `summary-report.txt`.
